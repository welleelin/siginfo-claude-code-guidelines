## 🚀 快速预览

### 方式 1: 使用预览脚本 (推荐)

```bash
# 进入展示目录
cd docs/capability-showcase

# 运行预览脚本
./preview.sh

# 选择要预览的页面:
# 1 - Web UI 展示页面（含专家矩阵）
# 2 - Web PPT 幻灯片
# 3 - 两者都打开
```

### 方式 2: 直接用 Python 启动

```bash
# 进入展示目录
cd docs/capability-showcase

# 启动 HTTP 服务器
python3 -m http.server 8080

# 在浏览器中打开:
# Web UI:       http://localhost:8080/index.html
# 专家矩阵：    http://localhost:8080/agents.html
# Web PPT:      http://localhost:8080/presentation.html
```

### 方式 3: 直接打开文件

```bash
# macOS
open index.html
open presentation.html

# Linux
xdg-open index.html
xdg-open presentation.html

# Windows
start index.html
start presentation.html
```

## 📝 自定义修改

### 修改配色

在 HTML 文件中找到 `:root` 变量定义：

```css
:root {
  --primary: #2563EB;      /* 主色 */
  --success: #10B981;      /* 成功色 */
  --warning: #F59E0B;      /* 警告色 */
  /* ... */
}
```

### 修改内容

直接编辑 HTML 文件中的文本内容。

### 添加新页面/幻灯片

- **Web UI**: 复制 `<section>` 块，修改 ID 和内容
- **Web PPT**: 复制 `<div class="slide">` 块，修改内容
