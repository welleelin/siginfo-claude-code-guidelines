## 🔧 自动化检查

### 构建检查

```bash
# 检查构建
npm run build

# 检查类型
npm run type-check

# 检查 Lint
npm run lint
```

### 测试检查

```bash
# 运行所有测试
npm test

# 检查覆盖率
npm test -- --coverage

# 运行 E2E 测试
npx playwright test
```

### 代码审查

```bash
# 自动代码审查
/code-review

# 安全审查
/security-review
```

## 📊 质量指标

| 指标 | 最低要求 | 推荐值 |
|------|---------|--------|
| 测试覆盖率 | 80% | 90%+ |
| 构建时间 | <5 分钟 | <2 分钟 |
| 测试通过率 | 100% | 100% |
| 代码审查 | 无 CRITICAL/HIGH | 无问题 |
