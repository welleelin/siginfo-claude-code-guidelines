📋 Standard Flow

适合中等任务（复杂度 4-6），预计时间 2-8小时

2

4

5-8

🏢 Enterprise Flow

适合复杂任务（复杂度 7-10），预计时间 1-4周

1
Analysis（分析阶段）

头脑风暴 → 领域研究 → 市场研究 → 产品简报

2
Planning（规划阶段）需确认

创建PRD → UX设计 → 方案确认

3
Solutioning（方案设计）

架构设计 → Epic/Story分解 → 实施就绪检查

4
Implementation（实现）

Sprint规划 → 多Story并行开发 → 代码审查

5
Quality Assurance（质量保障）

API完整性 → E2E测试 → 安全审查 → 质量门禁

引擎架构

⚙️ WorkflowEngine

智能工作流引擎，自动评估任务复杂度并选择最适合的工作流。

const engine = new WorkflowEngine(); const result = await engine.execute({ task: "实现用户登录功能", complexity: await engine.assessComplexity(task) }); // 自动选择 Quick Flow / Standard Flow / Enterprise Flow

🔒 QualityGateEngine

四层质量门禁引擎，确保代码质量和安全性。

const gates = [ 'code-quality', // 代码质量门禁 'api-completeness', // API完整性门禁 'security', // 安全性门禁 'final-quality' // 最终质量门禁 ]; const result = await engine.runAllGates(gates);

🚀 快速开始

5 分钟体验核心流程

# 1. 克隆本项目 git clone https://github.com/your-org/siginfo-claude-code-guidelines.git cd siginfo-claude-code-guidelines # 2. 运行安装脚本 ./scripts/install.sh # 3. 初始化记忆系统 ./scripts/init-memory.sh # 4. 开始第一个任务 /plan "实现用户登录功能"
