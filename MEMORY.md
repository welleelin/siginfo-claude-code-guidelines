## ⚠️ 经验教训

### 2026-03-06 - macOS sed 兼容性问题
- **问题**：init-memory.sh 中 sed 命令在 macOS 上报错
- **原因**：macOS sed 与 GNU sed 语法差异
- **解决方案**：使用 perl -pe 替代 sed 进行模板变量替换
- **预防措施**：跨平台脚本优先使用 perl 或 python
